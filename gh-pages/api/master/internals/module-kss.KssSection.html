(private) _propertyHelper(propertyName, objectConstructor, properties) → {KssSection|Array|KssModifier|false}

Gets or adds nested objects of the section.

A common helper for .modifiers() and .parameters() methods.

Different types of arguments for properties will yield different results:

  • Object|Array: If the value is an array of objects or an object, the properties are added to this section.
  • undefined: Pass nothing to return all of the section's properties in an array.
  • integer: Use a 0-based index to return the section's Nth property.
  • string: Use a string to return a specific modifier by name.
Parameters:
Name Type Description
propertyName string

The name of property in KssSection.
objectConstructor Constructor

The constructor function for the type of object the property is.
